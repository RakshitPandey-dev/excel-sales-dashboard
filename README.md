# 📊 Sales Performance Dashboard | Microsoft Excel

An interactive **Sales Performance Dashboard** built in **Microsoft Excel** to analyze sales data and present key business insights through dynamic visualizations and KPI cards.

This project demonstrates my ability to organize raw data, create Pivot Tables, build interactive charts, and design a clean dashboard for business reporting.

## 📊 Dashboard Features

### KPI Cards
- Total Sales
- Total Orders
- Total Units Sold
- Highest Selling Product
- Highest Performing Region

### Interactive Visualizations
- 📌 Category-wise Sales Distribution (Donut Chart)
- 📌 Salesperson-wise Sales Comparison (Column Chart)
- 📌 Daily Sales Trend (Line Chart)

### Interactive Slicers
Users can filter the dashboard by:

- Region
- Product
- Salesperson
- Category

All charts and KPIs update dynamically based on the selected filters.
